# OpenClaw Workspace

Workspace configuration, scripts, and tools for OpenClaw personal assistant on Orange Pi 5.

## Structure

```
.openclaw/
├── workspace/
│   ├── README.md              # Technical setup guide
│   ├── TOOLS.md              # Environment-specific notes
│   ├── IDENTITY.md            # Assistant persona
│   ├── USER.md               # User profile
│   ├── AGENTS.md             # Agent configuration
│   ├── MEMORY.md             # Long-term memory (gitignored)
│   ├── memory/               # Daily notes (gitignored)
│   ├── media/                # Images & attachments (gitignored)
│   ├── tools/                # Helper scripts
│   │   ├── ddg_search.py
│   │   ├── ihsg_data.py
│   │   ├── fetch.py
│   │   ├── tts.py
│   │   ├── ocr.py
│   │   └── image_gen.py
│   ├── stock_analysis.py      # Stock analysis scripts
│   ├── comprehensive_analysis.py
│   └── .gitignore           # Git ignore rules
└── config.json               # Configuration (gitignored)
```

## What's Included

- **Setup Guides:** README.md with complete technical documentation
- **Scripts:** Stock analysis, web fetching, TTS, OCR, image generation
- **Tools:** Helper scripts for common tasks
- **Documentation:** IDENTITY.md, USER.md, AGENTS.md

## What's Excluded (via .gitignore)

- **Sensitive Data:** config.json, API keys, bot tokens
- **Personal Data:** MEMORY.md, memory/ directory
- **Media Files:** images, PDFs, attachments
- **Temporary Files:** logs, cache, temp files

## Installation

### Clone & Setup

```bash
# Clone repository
git clone https://github.com/yourusername/openclaw-workspace.git ~/.openclaw/workspace

# Setup asdf (Python, Node.js)
asdf install python latest
asdf install nodejs lts
asdf global python latest nodejs lts

# Install Python packages
pip install yfinance pandas numpy pyttsx3 duckduckgo-search playwright pytesseract Pillow

# Install system packages
sudo apt install espeak-ng tesseract-ocr tesseract-ocr-indo

# Create tools directory
mkdir -p ~/.openclaw/workspace/tools
cd ~/.openclaw/workspace/tools
```

### Configure OpenClaw

```bash
# Configure web search (Brave API)
openclaw configure --section web

# Setup Telegram (update config.json manually or via wizard)
# Bot tokens should be kept private
```

## Usage

### Stock Analysis

```bash
# Run comprehensive analysis
python3 ~/.openclaw/workspace/stock_analysis.py

# IHSG data
python3 ~/.openclaw/workspace/tools/ihsg_data.py "^JKSE"
```

### Web Search

```bash
# DuckDuckGo search
python3 ~/.openclaw/workspace/tools/ddg_search.py "IHSG hari ini"
```

### TTS

```bash
# Text-to-speech
python3 ~/.openclaw/workspace/tools/tts.py "Halo ini tes"
```

### OCR

```bash
# Text recognition
python3 ~/.openclaw/workspace/tools/ocr.py "/path/to/image.jpg"
```
